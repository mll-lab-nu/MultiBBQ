# Templates

Source templates and vocabulary for **dataset construction**. These are inputs to
[`../../../notebooks/gen_template.ipynb`](../../../notebooks/), not to evaluation; the
evaluation sets live in [`../../metadata/`](../../metadata/).

| File | Contents |
|---|---|
| `new_templates_Age_v2.csv` | Age-category context/question templates. |
| `new_templates_Gender_identity.csv` | Gender-identity templates. |
| `new_templates_Race_ethnicity_v3.csv` | Race/ethnicity templates. |
| `new_templates_Religion_v2.csv` | Religion templates. |
| `vocabulary.csv` | Demographic-group names, nouns, and lexical-diversity terms slotted into the templates. |
